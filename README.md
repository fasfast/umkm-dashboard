# UMKM Dashboard (Python + Streamlit)

Web dashboard untuk mengolah dan menganalisis laporan penjualan marketplace (Shopee, Tokopedia, dll).

## Features
- Upload multiple CSV files
- Data cleaning & formatting
- Grouping by platform & date
- Daily omzet calculation
- Platform omzet summary
- Export to Excel
- Interactive charts

## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
- XlsxWriter

## Purpose
Project ini dibuat untuk membantu UMKM memproses laporan penjualan secara otomatis dan menghemat waktu.
