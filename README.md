# 📊 Dashboard Laporan UMKM — Multi Marketplace

> **Gabungkan laporan Shopee & Tokopedia secara otomatis. Analisis omzet, produk terlaris, dan export ke Excel dalam 1 klik.**

🔗 **[➡️ Coba Langsung (Live Demo)](https://umkm-dashboard.streamlit.app)**

---

## 🎯 Cocok untuk Siapa?

- Pemilik toko online di **Shopee & Tokopedia**
- Admin yang capek rekap laporan manual tiap hari
- UMKM yang ingin laporan penjualan lebih rapi & cepat

## 🚀 Cara Pakai

1. Buka **[umkm-dashboard.streamlit.app](https://umkm-dashboard.streamlit.app)**
2. Download contoh file CSV dari sidebar (opsional, untuk coba demo)
3. Upload file CSV laporan **Shopee** dan **Tokopedia**
4. Lihat dashboard otomatis terbentuk 🎉
5. Klik **Download Laporan Excel** untuk simpan hasil

---

## 📋 Format File CSV

Pastikan file CSV kamu punya kolom berikut:

```
tanggal, produk, qty, harga, total
2024-01-01, Kaos Polos, 2, 85000, 170000
2024-01-01, Celana Jeans, 1, 120000, 120000
```
